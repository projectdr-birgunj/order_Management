Privacy Policy

Data Collection

We collect data such as order details, table assignments, user roles, and transaction information to enhance order accuracy and billing efficiency.

User Roles and Permissions

Each role—waiter, chef, and cashier—has specific access, ensuring users only view the information needed for their tasks.

Data Use

Collected data is used to manage orders, process transactions, and improve service, strictly within restaurant operations.

Data Security

We implement strict measures to protect stored data from unauthorized access, with secure protocols for handling order and transaction information.

Payment Processing

Payment data is handled securely via a third-party provider through QR codes, ensuring customer details are not stored in the app.

Data Sharing

Order and transaction data is not shared with outside parties except when necessary for secure payment processing.

Data Retention

Data is stored only as long as needed for operations, with regular reviews to delete outdated information.

User Consent

Using the app implies consent to data practices, and users have the right to access or update information related to their role.

Access and Control

Users may request access or corrections to their personal data relevant to their role, ensuring transparency and accuracy.

Policy Changes

We will notify all staff promptly of updates to the privacy policy to keep everyone informed of new terms.
